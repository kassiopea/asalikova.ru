Шпаргалка. Команды Linux

Здесь я выписала основные команды для терминала Linux. Один из способов открыть терминал: комбинация клавиш ctrl + alt + T. В настоящую консоль (без графической оболочки) можно войти через Ctrl + Alt + F1 (последняя клавиша определяет номер терминала, можно использовать от одного до шести, то есть от F1 до F6. А выйти из нее командой клавиш Ctrl + Alt + F7. Эта шпаргалка будет пополняться новыми командами в процессе освоения мною терминала и консоли.

Копировать - Ctrl + Insert

Вставить - Shift + Insert

Помощь в автозаполнении - tab

Последняя набранная команда - стрелка вверх на клавиатуре

history - история команд, набранный в терминале

ls - содержимое каталога

xdg-open имя файла - открывает файл в программе. Я обычно открываю таким образом сохраненный плейлист в VLC.

df Показывает занятое пространство на диске (по умолчанию в байтах)

df -h Показывает занятое пространство на диске в гигабайтах

apt-get update получить обновление списков пакетов (этап перед установкой пакетов)

apt-get install package_name установить или обновить пакет, где package_name имя пакета

pkill [название приложения]

(или process kill) завершает запущенный процесс. Часто используется в случаях, когда приложение не отвечает

top Команда выводит самые ресурсоемкие процессы в системе

cd - домашний каталог

cd "путь" - переходит в директорию "путь". Принимает как относительный, так и абсолютный.

Относительный путь:

cd music

Абсолютный путь:

cd "/serials/2017/Анатомия Грей"

В случае, когда есть плохие символы в названии директории, путь нужно взять в кавычки (cd "Мои документы"). Или обратный слеш перед пробелами.

cd - (cd c тире) означает переход в предыдущую дерикторию

pwd Путь к текущему каталогу

mkdir -p [путь до директории] создать директорию в указанном месте

shutdown выключение компьютера. Используется с параметрами

$ sudo shutdown [опции] [время] [сообщение]

Где опции:

$ sudo shutdown -h now

Выключить компьютер сейчас

$ sudo shutdown -h +5

Где +5 выключение через определенное количество минут, в данном случае, выключкение через 5 минут.

$ sudo shutdown -c

Отмена предыдущей команды выключения

$ sudo shutdown -h 00:00

Выключение компьютера к определенному времени, где 00:00 - часы и минуты, заданные пользователем

А poweroff команда используется для выключения компьютера, но при этом не выполняются подготовительные действия перед выключением. Просто отключается питание

$ sudo poweroff

touch [index.html] создать новый файл. В квадратных скобках указан пример названия файла
